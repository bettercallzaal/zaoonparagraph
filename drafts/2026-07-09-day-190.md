day 190 of building on zabal gamez.

three fixes landed and none of them look like much until you notice what they quietly stopped.

the video editor had an auth check that failed open instead of closed. wrong door left unlocked instead of locked. closed the gap before anyone walked through it. shipped as [fix auth fail-open and missing upload validation](https://github.com/bettercallzaal/ZAOVideoEditor/pull/48).

a 54 minute recording ran into a whisper transcription bug that spun into a repetition loop and quietly ate the back half of the transcript. no crash no error just wrong words standing in for the real ones. fixed it inside [make audio-only recordings a first-class input](https://github.com/bettercallzaal/ZAOVideoEditor/pull/49) which also swapped the render check from comparing pixels to reading the actual words. it reads the record not your word.

and this newsletter got its own pipe. a direct line from a markdown file straight into paragraph as a draft. the api key never touches a chat window and never touches a saved file. one hidden terminal prompt and nothing else. shipped as [wire the paragraph draft api pipe](https://github.com/bettercallzaal/zaoonparagraph/pull/1). this issue is the first thing that ran through it.

one line of auth logic. one transcription loop. one script. small builds. the quiet work compounds.

coming next. the same pipe that just carried this issue into paragraph will carry the socials out once it publishes. firefly then the group chats then the rest. one clean flow end to end instead of five manual ones.
